## Requirements

`LimitCellSelect` is written in Swift 3. Compatible with iOS 9.0+

## Installation

### Cocoapods

LimitCellSelect is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LimitCellSelect'
```
